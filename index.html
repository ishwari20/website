Summer of Learning?

Summer of Learning Program - CCEW is a student initiative to encourage effective online learning through massive open online courses (MOOCs) via a mentor-mentee system, with the secondary aim to create a healthy academic peer culture among CCEW students.

Course List

Computer/IT

Web Development

The Complete 2021 Web Development Bootcamp

The only course you need to learn web development. There are a lot of options for online developer training, but this course is without a doubt the most comprehensive and effective on the market. This is the only complete beginner full-stack developer course that covers NodeJS.

Computer/IT

Hardware Explanation

CCNA/CCNP

A must have for the I.T. branch, it is provided by the dept. anyway. So, the ones who don't get this experience are the other branch students.Includes hardware explanation in dept plus assessments and tools to be introduced to like packet tracer and wireshark.

Computer Science

Python

Joy of Computing using Python

Good course for beginners to python, may be a bit of a drag in the beginning but continuous practice will yield good results. Also, this is an elective for the current I.T. TY and COMP TY.

Computer Science

Data Science

Introduction to Computational Thinking and Data Science

Get handy with Python 3 and use it to solve complex real world problems. Learn statistical concepts and computational thinking. Slightly advanced. Those without familiarity with Python language will have to take extra efforts in learning it.

Computer Science

Data Science

Applied Data Science with Python Specialization (Course 1 and Course 2)

The first course is a short introductory course on using Python for data science. Data science and Python are sought-after skills in whichever field you go. Build on the skills from the first course and apply machine learning in Python to solve problems.

Computer Science

Andrew NG - ML

Machine Learning

Perfect course for understanding theory as well as implementation aspects of machine learning. Covers the math behind the ML, thus helping understand the inner workings of machine learning algorithms. Implementation in MATLAB. Useful for any project involving machine learning. Some assignments are slightly tough. Requires knowledge of basic programming concepts.
